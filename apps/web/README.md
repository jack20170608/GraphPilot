# GraphPilot Web Console

This app will host the GraphPilot web console.

Planned stack:

- Next.js
- TypeScript
- Tailwind CSS
- Zustand
- React Flow
- TanStack Query
- TanStack Table
- shadcn/ui

Planned areas:

- Workflow list and detail pages
- DAG editor and viewer
- Workflow run monitoring
- Task instance logs
- Schedule management
- Settings and administration
